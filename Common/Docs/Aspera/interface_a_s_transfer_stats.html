Method Documentation

- (NSString *) averageTimeRemaining

Average time remaining, in seconds. Formatting is the same as that used for timeElapsed

- (NSString *) averageTransferRate

Average transfer rate, as a string. Rate is returned in the largest possible metric unit (up to Tb/s). Rates larger or equal to 1 Kb/s are returned with accuracy to two decimal places. Examples:

  • 161 b/s
  • 314.15 Kb/s
  • 27.18 Mb/s

Note that in this context 1Kb = 1,000b, 1Mb = 1,000Kb etc.

- (NSString *) instantaneousTransferRate

Instantaneous transfer rate, as a string. Rate is returned in the largest possible metric unit (up to Tb/s). Rates larger or equal to 1 Kb/s are returned with accuracy to two decimal places. Examples:

  • 161 b/s
  • 314.15 Kb/s
  • 27.18 Mb/s

Note that in this context 1Kb = 1,000b, 1Mb = 1,000Kb etc.

- (NSString *) timeElapsed

Returns the elapsed transfer time as a string. The time will be given in the form 'Xm Ys' for X minutes and Y seconds. The minutes will be dropped if elapsed time is under one minute. For elapsed time under one second, seconds will be given in fractional form with accuracy to two decimal places (e.g. "0.35s")

- (NSString *) totalRemaining

Total bytes remaining to transfer, as a string. Formatting is the same as that used for totalSize

- (NSString *) totalSize

Total size of transfer, as a string. The size is returned in the largest possible "metric" unit (up to TB). Sizes larger than or equal to 1KB are returned with accuracy to two decimal places. Examples:

  • 161 B
  • 314.15 KB
  • 27.18 MB

Note that in this context 1KB = 1,024B, 1MB = 1,024KB, etc.

- (NSString *) totalTransferred

Total bytes transferred in this session, as a string. Note that the receiver may have a larger portion of the payload than represented by this value if the session is a continuation of a previously interrupted transfer. Formatting is the same as used for totalSize.

- (NSString *) totalWritten

Total bytes written at receiving end, as a string. This quantity may be larger than the number of bytes transferred in this session if this session is a continuation of a previously interrupted transfer. Formatting is the same as used for totalSize.
