Present a modal view with a prepared email containing diagnostic information and the log files as attachments. Recipients should be a semicolon separated list of email addresses. The model view is presented by the given view controller. An error message will be shown to the user if no email accounts are configured. See logFiles (AsperaMobile) for direct access to the log files
