- (void) exploreDropboxesForAccount: (ASFaspexAccount *)  account

n Initiate dropbox exploration for the given faspex account. The account should be one that supports the dropbox listing service. Dropbox exploration means: each dropbox listed in the dropbox list service for the given account has its information retrieved and the corresponding FaspexDropbox object in core data is updated/created. The dropboxes attribute for the given account is made to updated to match the list returned by the service. (Note that an account subscribed to N dropboxes needs N+1 web requests to accomplish this task: one to retrieve the list of dropboxes and one per dropbox to get its information).
