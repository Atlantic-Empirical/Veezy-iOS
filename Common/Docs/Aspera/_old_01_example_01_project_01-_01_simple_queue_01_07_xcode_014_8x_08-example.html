Old Example Project - SimpleQueue (Xcode 4.x)
Note
This sample project appeared in previous releases of the SDK. It is fully functional and demonstrates correct usage of the SDK but it is more suitable for use with earlier versions of Xcode (e.g., Xcode 4).

An XCode project that provides a complete working example of leveraging the Aspera iOS SDK.

This project implements a minimal queue interface that allows uploading and downloading from one (hard-coded) Aspera server.

Download the project files

To build this project:

  • download, extract and open the project
  • follow the SDK installation instructions to add the Aspera iOS SDK to this project (note that the necessary frameworks have already been added)
  • edit ApplicationSettings.m and replace the placeholder account configuration values with your destination server parameters
  • Make sure you have a valid provisioning profile if building for an iOS device, or select to build for the iOS simulator
  • build and run
