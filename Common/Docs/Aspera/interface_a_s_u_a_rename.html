- (void) startWithSuccess: (NSDictionary *)  success
failure: (NSError *)  failure 

Start rename request for one or more file/directory renames. Success block is executed upon request completion. You must parse the response NSDictionary for errors with individual rename errors. For rename requests with multiple renames some may succeed and others may fail. The NSDictionary will have a key "paths" containing an NSArray of successful or failed renames (NSDictionaries). Check each object of the array (NSDictionaries) if they have the key "error".

- (NSString *) password [read, write, assign]

The password to use for HTTP Basic authentication. If either of the user or password are nil, no authentication headers are included in the request.

- (NSInteger) port [read, write, assign]

The port on the server where the Unified App API is listening. The default port settings for the Unified App API are: 80 for plaintext communication and 443 using ssl encyption.
