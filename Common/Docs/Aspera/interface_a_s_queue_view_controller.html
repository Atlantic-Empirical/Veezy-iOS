ASQueueViewController Class Reference

Detailed Description

A view controller that provides an interface to examine and manipulate transfers in the queue. This view controller can be used in any application to easily provide a UI for transfer management. See Queue View.

#import <ASQueueViewController.h>

Inherits UITableViewController, <ASTransferQueueDelegate>, <UITableViewDelegate>, and <ASQueueViewControls>.
