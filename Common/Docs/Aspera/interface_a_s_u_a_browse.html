Property Documentation

- (BOOL) allowSelfSignedSSL [read, write, assign]

Determines if self-signed SSL certificates are allowed. If set to NO, a request that encounters a self-signed SSL certificate will fail unless the host is listed in the list trusted hosts. Defaults to NO.

- (NSString *) apiUrl [read, write, assign]

The API URL as returned by the Shares API query call.

- (NSArray *) basenames [read, write, assign]

The property "basenames" is an NSArray containing NSStrings, only the files and folders whose basenames matches those specified in "basenames" are returned from the server. E.g. "*.jpg", "file.*"

- (BOOL) caseSensitive [read, write, assign]

Boolean property indicating whether any of the above parameters and filters are case sensitive. Default value is "false".

- (NSInteger) count [read, write, assign]

An NSInteger indicating how many items should be returned. The "count" value cannot exceed 100. The default value is 100. If the directory you are browsing contains more than 100 items you must perform multiple separate browse request. E.g. A directory containing 150 files can be browsed as such:

  1. A browse with "skip"=0 and "count"=75 (returns items 1-75).
  2. A second browse with "skip"=75 and "count"=75 (returns items 75-150).
- (NSString *) host [read, write, assign]

Hostname of the server running the Unified App API. Can be IP address of hostname.

- (NSDate *) mTimeMax [read, write, assign]

If property is set, only files and folders whose modified date occures prior to mTimeMax are returned from the serve.

- (NSDate *) mTimeMin [read, write, assign]

If property is set, only files and folders whose modified date occurs after mTimeMin are returned from the server.

- (NSString *) password [read, write, assign]

The password to use for HTTP Basic authentication. If either of the user or password are nil, no authentication headers are included in the request.

- (NSString *) path [read, write, assign]

Specifies the path to the directory you wish to browse.

- (NSArray *) paths [read, write, assign]

The property "paths" is an NSArray containing NSStrings, only the files and folders whose paths matches the format specified in "paths" are returned from the server. E.g. "b/c*";

- (NSInteger) port [read, write, assign]

The port on the server where the Unified App API is listening. The default port settings for the Unified App API are: 80 for plaintext communication and 443 using ssl encyption.

- (NSString *) root [read, write, assign]

The path components to the root directory. (e.g. if your server us running on "www.aspera.us/path/to/root" this property should be "/path/to/root")

- (NSString *) scheme [read, write, assign]

URL scheme of the server. (e.g. "http" or "https")

- (NSInteger) sizeMax [read, write, assign]

Only include files whose size is less than or equal to "sizeMax" (in bytes). Default value is -1 (unused).

- (NSInteger) sizeMin [read, write, assign]

Only include files whose size is greater than or equal to "sizeMin" (in bytes). Default value is -1 (unused).

- (NSInteger) skip [read, write, assign]

An NSInteger indicating how many items are skipped. E.g. If skip is set to 50. The server will return files and folders starting from the 51st item. The default value is 0 (no skip).

- (NSString *) sort [read, write, assign]

Sort property to specify response order. Can be sorted by size, size descending, modified time, filename, filename descending, type.

- (NSArray *) types [read, write, assign]

The property "types" is an NSArray containing NSStrings, only the files and folders whose type matches those specified in "types" are returned from the server. The possible types can be: "file", "directory", or "symbolic_link".

- (NSString *) user [read, write, assign]

The user to use for HTTP Basic authentication. If either of the user or password are nil, no authentication headers are included in the request.
