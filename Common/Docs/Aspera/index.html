iOS Mobile SDK

The Aspera Mobile SDK for iOS provides a queue-based API for initiating, managing and monitoring FASP transfers on iOS devices.

The Mobile SDK is part of the Aspera ecosystem of products and applications can leverage it to integrate effortlessly with other Aspera software such as Enterprise Server, Console and Orchestrator.

Note
v3.0.0 of the Aspera Mobile SDK supports iOS 7.0 and the armv7 and armv7s architectures but not the arm64 architecture. Please consult the installation instructions for information on how to disable this architecture in your project (specifically step 6)
