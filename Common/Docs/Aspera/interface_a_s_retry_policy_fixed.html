ASRetryPolicyFixed Class Reference

Detailed Description

A Retry policy that allows a fixed number of retries before disallowing further attempts. The waitDuration is identical for each retry. When the policy parameters are reset numTries is set to zero and maxTries new retries are allowed.

#import <ASRetryPolicyFixed.h>
